## 📱 Про проект

**AssistentNUOS** - це сучасний прогресивний веб-додаток (PWA) створений спеціально для студентів Національного університету кораблебудування імені адмірала Макарова. Додаток надає зручний та швидкий доступ до всієї необхідної інформації про університетське життя.

## ✨ Основні функції

### 🎯 Розділи додатку:
- 🏠 **Головна сторінка** - Швидкий доступ до всіх функцій
- 👥 **Про нас** - Команда розробників та контакти
- 🏛️ **Університет** - Повна інформація про НУК
- 📅 **Розклад** - Розклад дзвінків та академічний календар
- 👔 **Керівництво** - Керівний склад університету
- 🏛️ **Студентське самоврядування** - Органи студентського представництва

### 🚀 PWA можливості:
- 📱 **Установка на пристрій** - Працює як нативна програма
- 🌐 **Офлайн режим** - Повний доступ без інтернету
- ⚡ **Швидке завантаження** - Миттєвий старт завдяки кешуванню
- 📱 **Адаптивний дизайн** - Ідеально на всіх пристроях
- 🔔 **Push-сповіщення** - Важливі новини та оновлення

## 🚀 Швидкий старт

### Крок 1: Завантаження
```bash
git clone https://github.com/IZenApp/AssistentNUOS.git
cd AssistentNUOS
```

### Крок 2: Запуск сервера
```bash
# 🐍 Python (рекомендовано)
python3 -m http.server 8080

# 📦 Node.js
npx serve -l 8080

# 🔧 PHP
php -S localhost:8080
```

### Крок 3: Відкриття
Перейдіть за адресою: **http://localhost:8080**

## 📱 Встановлення PWA

1. 🌐 Відкрийте сайт у браузері
2. 📲 Натисніть "Встановити" в адресному рядку
3. ✅ Підтвердіть встановлення
4. 🎉 Користуйтесь як нативним додатком!
